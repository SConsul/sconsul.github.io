CryptoSuite

This was done as a course project of of the introductory programming course. We were tasked to create a collection of C++ programs that in conjection could handle the basic tasks in cryptography.

  • Encrypted and Decrypted alphanumeric messages using RSA Algorithm
  • Implemented Baby Step Giant Step Algorithm to find discrete logarithms
  • Binary and Prime Field Operations
  • Modulo Arithmetic - addition, multiplication, inverse, exponentiation
